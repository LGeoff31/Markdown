# Markdown

# H1 text is creaed using a single hashtag
## H2 text is created using a double hashtag
### And so on...

A paragraph tag doesn't require any hashtags.

**Bolding text requires two surrounding asteriks on both sides**

_italize requires an underscore on both sides_

A [link](http:youtube.com) requires the display in square brackets and the url following in parenthesis

A ![image](http://google.com/image/logo.gif) requires the same thing but prefixed with a !
